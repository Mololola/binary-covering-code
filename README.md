# binary-covering-code
Binary covering-code–based encoding strategy for compressing 20-bit solutions into 10-bit messages with a provable worst-case guarantee, using Hamming-space analysis and block-wise decoding.


## Files
- `exam_cheat_code.py` – Implementation of `encode()` and `decode()`.
- `exam_tester.py` – Exhaustive local tester (provided by the module).
- `explanation.pdf` – One-page explanation of the method.
- `impossibility.pdf` – Optional impossibility upper-bound argument.

## How to run
Make sure you have Python 3 installed.

```bash
python exam_tester.py
